# Typography

cytario uses **Montserrat** as its primary typeface. It is a geometric sans-serif that works well at both display and body sizes, and carries a modern, approachable tone appropriate for medical technology.

## Line heights

| Token | Value | Usage |
|-------|-------|-------|
| `--line-height-tight` | 1.25 | Headings, display text, compact UI labels |
| `--line-height-normal` | 1.5 | Body text, paragraphs, form labels |
| `--line-height-relaxed` | 1.625 | Long-form content, documentation prose |

---

## Usage guidelines

**Headings** use semibold or bold weight with tight line height. Color: `--color-text-primary` (neutral-900).

**Body text** uses regular weight at base size (16px) with normal line height. Color: `--color-text-secondary` (neutral-600) for supporting content, `--color-text-primary` for main content.

**Captions and labels** use sm (14px) or xs (12px) at medium weight. Color: `--color-text-tertiary` (neutral-400) for least-prominent text.

**Never** use light (300) weight below 18px -- it becomes hard to read on lower-resolution screens.
