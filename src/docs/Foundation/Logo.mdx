# Logo

The cytario logo is the most recognizable element of the brand. Use it consistently and with care.

---

## React component

Use the `<Logo>` React component in application UIs (headers, sidebars, auth screens). It renders an inline SVG with CSS animations, no image requests, and **automatically follows the active theme** — purple wordmark on light backgrounds, white wordmark on dark backgrounds.

```tsx
import { Logo } from "@cytario/design";

<Logo />              // theme-driven (purple on light, white on dark)
<Logo scale={1.4} />  // scale factor
```

### Theme-driven colors

The logo uses `--logo-fill` and `--logo-highlight` CSS variables that switch automatically with `[data-theme="dark"]`. No prop needed — the logo adapts to its context.

### On brand purple

The logo does not automatically detect non-theme backgrounds. On a brand-purple background, pass `color="#ffffff"` to force a white wordmark:

```tsx
<Logo color="#ffffff" />
```

<div style={{ borderRadius: 12, padding: 48, textAlign: "center", background: "#5c2483", marginTop: 24, display: "flex", flexDirection: "column", alignItems: "center" }}>
  <Logo scale={3} color="#ffffff" />
</div>

### Escape hatch (custom colors)

Pass `color` and `highlightColor` to override the theme-driven values. Set both to the same value for monochrome.

```tsx
<Logo color="#000000" highlightColor="#000000" />  // monochrome black
<Logo color="#ffffff" highlightColor="#ffffff" />  // monochrome white
<Logo color="#dc2626" highlightColor="#2563eb" />  // custom
```

## Brand guidelines

**Clear space:** Maintain minimum clear space equal to the height of the "c" in "cytario" around the logo.

**Minimum sizes:** With tagline — 120 px screen / 30 mm print. Standard — 48 px screen / 12 mm print.

**Do:**
- Use the provided component or SVG files — never recreate the logo
- Place on solid, uncluttered backgrounds
- Let the logo follow the theme automatically
- Maintain the correct aspect ratio at all times

**Don't:**
- Stretch, rotate, or skew the logo
- Change the logo colors (use the escape hatch only when strictly necessary)
- Place on busy or low-contrast backgrounds
- Add drop shadows, outlines, or effects
- Crop or partially obscure the logo

**Registered trademark:** The cytario name and logo carry the &reg; mark. It is already incorporated into the logo component and SVG files.

**Regulatory use:** For regulatory submissions, use the monochrome black variant (`cytario-logo-mono-black.svg`) to ensure legibility in grayscale printing and archival formats.
