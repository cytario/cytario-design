# Spacing

cytario uses a **4px base unit** spacing scale. Every margin, padding, and gap in the system is a multiple of 4px, producing a consistent visual rhythm across all screens.

## Usage guidelines

- Use spacing tokens exclusively. Hard-coded pixel values break when the scale is adjusted.
- Inner padding of interactive elements (buttons, inputs) typically uses **spacing-2** (8px) to **spacing-3** (12px).
- Gap between form fields: **spacing-4** (16px).
- Gap between major page sections: **spacing-12** (48px) or **spacing-16** (64px).
- When in doubt, choose the smaller value. Pathologists work on information-dense screens -- space is at a premium.
