# cytario design system

Single source of truth for the cytario corporate identity -- from brand foundation to production-ready components.

This portal documents every element of the cytario visual language: logos, color palette, typography, spacing, and the component library built on top of them. It is the reference for designers, developers, and anyone producing cytario-branded materials.

## How to use this portal

**Designers** -- Use foundation pages as the definitive color, type, and spacing reference. Component stories show every interactive state.

**Developers** -- Copy token values directly from the color and spacing pages. Every component story includes working code.

**Marketing and leadership** -- The Guidelines section contains approved claims, brand voice rules, and logo usage rules.
