# Typography

cytario uses **Montserrat** as its primary typeface. It is a geometric sans-serif that works well at both display and body sizes, and carries a modern, approachable tone appropriate for medical technology.

## Type scale, font weights, and line heights

cytario uses **Tailwind v4's defaults** for font size, font weight, and line height — no custom overrides. This keeps text accessible (rem-based, scales with browser font-size) and avoids reinventing what Tailwind already does well.

See the Tailwind docs for the full reference:

- [Font size](https://tailwindcss.com/docs/font-size) — `text-xs` through `text-9xl` (rem-based, with paired line-heights)
- [Font weight](https://tailwindcss.com/docs/font-weight) — `font-thin` through `font-black`
- [Line height](https://tailwindcss.com/docs/line-height) — `leading-none` through `leading-loose`

---

## Usage guidelines

**Headings** use semibold or bold weight with tight line height. Color: `--color-foreground` (slate-900).

**Body text** uses regular weight at base size with normal line height. Color: `--color-muted-foreground` (slate-600) for supporting content, `--color-foreground` for main content.

**Captions and labels** use `text-sm` or `text-xs` at medium weight. Color: `--color-muted-foreground` for least-prominent text — the type hierarchy is two levels (foreground / muted-foreground); the old third tier was dropped because slate-400 fails WCAG AA on white.

**Never** use `font-light` below `text-lg` — it becomes hard to read on lower-resolution screens.
